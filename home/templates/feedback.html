Our pages

Landing Page

The Landing page is the page that is rendered to the user after login verification has been completed or after signup procedures... It is basically your home feed.
It has the following features:

  • Navigation Bar
  • Primary Sidebar
  • Secondary Sidebar
  • Post Field
  • Main content

Followers Page

The followers page is a destination to see requests that are sent to you from other cribbers. You also see a complete list of all your followers
It has the following features:

  • Navigation Bar
  • Primary Sidebar
  • Main content
  • Buttons to decline or accept requests

New Post

Its as cool as it sounds... The new post page allows you to create a caption-only post,image-only post,or a complete post.
It has the following features:

  • Navigation Bar
  • Primary Sidebar
  • Main content
  • Textbox for the caption
  • File select field to select the image
  • Create post button

Your crib

Your crib is just your own space. Or you can say your account blueprint...
It has the following features:

  • Navigation Bar
  • Primary Sidebar
  • Main content
  • Your own details
  • Your followers
  • And your posts

Notifications

It has the following features:

  • Navigation Bar

Bars Explained

The bars are just tools that enable swift and easy navigation throughout CribSpace...Learn more below

Navigation Bar

This is the most Important bar of all. It is located atop the main document. It is positioned in a way that you can still access it while you scroll. Read about its features below...

Nav-bar features

CribSpace Logo

Our official logo is the first entity to be spotted. It is on the left side of the navbar. Other than being the symbol of our platform,the logo can be clicked and will automatically redirect you to the home page or Landing page.

Search box

The second entity to the right is the search-box. With it you can search for any cribber in our database in realtime.It is fast,secure and very convenient.
Here's how to use:

  • Type a name into the box
  • Click the search icon on the left side of the box when done
  • Await results
  • If no results is seen,recheck spellings

Display Photo

The display photo is yet another powerful tool in the navigation bar. It is the dp of the currently logged-in user. But wait there's more..

When the photo i clicked,a hidden bar appears. It is the popup bar.

Click this > popup bar < to read about the popup bar.

Other Bars

Primary Sidebar

The primary sidebar contains links to virtually all pages and its available in all pages as well...
This are the components of the bar:

  • Home Page link
  • Followers page link
  • New post link
  • Your crib link
  • Notification page link
  • About CribSpace link
  • The footer text

Please note: The notification page link is not available in the primary sidebar if the page is the home page or the notifications page

Secondary Sidebar

The secondary sidebar is only available in the home/landing Page. Its purpose is to handle immediate notifications and recommend cribbers to follow while the user scrolls through the main content

This are the components of the bar:

  • Notification boxes
  • Notification page link
  • recommended cribbers box

Popup bar

The popup bar is by virtue,a bar that pops up when the user display photo on the navbar is clicked. It acts as an additional navigation tool on our platform.

Our dark-theme feature is available Here. It is in form of a button It can be clicked to toggle between dark-theme and light-theme.
